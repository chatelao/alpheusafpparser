# Contributing to Alpheus AFP Parser

Thank you for your interest in Alpheus AFP Parser! We welcome contributions from the community to help make this project even better.

## How to Contribute

### Reporting Bugs
If you find a bug, please open an issue on GitHub and provide as much detail as possible, including:
- A clear and descriptive title.
- Steps to reproduce the issue.
- Expected and actual results.
- Any relevant logs or screenshots.

### Requesting Features
If you have an idea for a new feature, please open an issue on GitHub to discuss it. We value community feedback and suggestions.

### Submitting Pull Requests
We welcome pull requests! To submit a pull request, please follow these steps:
1. Fork the repository and create a new branch for your changes.
2. Ensure your code follows the project's coding standards (use `./gradlew checkstyleMain`).
3. Write clear and concise commit messages.
4. Include tests for any new features or bug fixes.
5. Submit your pull request and wait for feedback.

## Coding Standards
- We use the Google Java Style Guide.
- All public classes, methods, and fields should have Javadoc comments.
- Follow existing naming conventions and project structure.

## Development Setup
- Java 21 or higher is required.
- Gradle is used for building and testing.
- Run `./gradlew test` to run all tests.

## Community
Join our community to discuss the project and get help:
- GitHub Issues
- GitHub Discussions

We look forward to your contributions!
